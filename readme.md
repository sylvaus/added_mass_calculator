# Added Mass Calculator
Simple implementation of the added mass calculation described in [link](http://digitool.library.mcgill.ca/R/?func=dbin-jump-full&object_id=139213&local_base=GEN01-MCG02)

## Installation
* git clone this repository 
* In this repository, run 
    * ``python setup.py install``
